一共安排了4天半在费城，其中4天的日程是要安排在会议上的。这是原来的安排，具体安排由于倒时差等原因，有所调整。

选择了一个充满阳光的上午，穿行在费城老城和富兰克林公园大道。
选择了一个雨后天晴的上午，穿行在宾大校园。
选择了一个消化不良的晚上，和当地人漫步在游人都不会去的老街。
选择了不能作更多选择的最后时刻，离开回家。

## 充满阳光的上午

{% img center http://thumbsnap.com/s/eBShysGp.png %}

房东对bnb 的解释似乎是Bike & Bed，虽然Breakfast 可以在冰箱里随便拿。多亏房东借自行车给我，让我找到了熟悉这座城市的最快方法。

网上的攻略说，秋天的费城是四季中最美的，看来我来的是多么幸运。我迎着阳光的方向前行，一方面是作为御寒，另一方面老城区在东面。街上的会时时见到鸽子、松鼠等出没。下面的视频抓拍于路肩的小水坑的卡车旁，起先是一只胖胖的麻雀发现了这里，然后就一只接着一只在这里玩起了水。

{% vimeo 145611745 %}


{% img left http://thumbsnap.com/s/tpNCwDuG.png 300 450  白色的教堂 %}
{% img right http://thumbsnap.com/s/ry25CaSQ.png 300 460  公园里的旧钢琴 %}

窄窄的街道两旁，错落着教堂、银行、故居等各式建筑，就算是民居，也是很别致的，只是和国内情况一样，城里大部分的房子都是租的，有钱人都住在郊外。
国家宪法中心，自由钟，富兰克林故居这些常规景点都骑行了一圈，遇到许多组团游玩的青少年，由于时间不多，没有仔细地参观。
最令我感到不同的，是这边的态度，街上碰到的人都出奇地热情，迎面走来的人脸上大多都是带着笑，陌生人之间打招呼似乎再正常不过，才小半个早上已经收到了8，9个 Have a nice day，我也渐渐地知道该怎么回复How you doing 这类的问候。文化之间的差异确实好有趣。

### Betsy Ross 之家

{% img right http://thumbsnap.com/s/rz6tkskw.png 400  家的外景 %}
{%blockquote Wikipedia  https://en.wikipedia.org/wiki/Betsy_Ross Besty Ross %}

Elizabeth Phoebe "Betsy" Ross (January 1, 1752 – January 30, 1836)，泛传根据家庭传统，认为她在1776年会见George Washington 将军后，制作了第一面美国国旗，George Washington 当时是陆军首席长官。她将国旗中的六角形改成了五角星，使其容易制作。然而，并没有直接的史料证据或口头记录来证实美国第一面国旗诞生的故事，并且这个故事第一次出现是在其孙子70年代所著的书中（中间间隔有一个世纪），再早就没有任何文献有这段故事的记载了。
{%endblockquote%}

不论Betsy Ross 是否真的制作了美国第一面国旗，她成了美国爱国主义教育的重点保护对象，她的故居被改造了，成为了一座博物馆，随身携带学生证的我便宜了5刀购得门票。

{% img center  http://thumbsnap.com/s/UGNOQPxB.png Betsty 原是一位裁缝专家 %}

参观了美国建国初期，也就大约乾隆时期美国建筑内部的样子，感觉住着会很挤。可以参观的地方共有三楼，还有地下室，内部饰品几乎是还原了当时的样子，可以看出她当时应该生活的不错。常驻的演员扮演Betsy 本人，讲解了如何用纸剪出一个五角星，并送了我一个剪下的星星。

### 费城艺术博物馆

{% img left http://thumbsnap.com/s/QIYyNG0c.png 500 费城艺术博物馆%}
阳光正好，不知不觉快要接近中午，简单吃了点东西，翻身上车，前往费城艺术博物馆。去往博物馆最快的方法就是走费城里最特殊的一条路，地图上唯一的斜线，鲜有的双向车道，富兰克林公园大街，路的中心是各个国家的国旗。不一会就到了博物馆，学生证又一次帮我省了5刀，在费城似乎大家都非常照顾学生群体。展馆很大，一个下午都是看不完的，所以这个票是两天有效，还可以用作附近罗丹艺术馆的入场凭证，只可惜后面没有时间再回来去罗丹艺术馆。

特馆里展出了当代美国艺术，有静物写生，现代艺术这些，发放的导游机可以在你有兴趣的展物面前，输入标签号码，听取讲解。明暗的变化足以骗过你的眼睛，但实在没有搞懂一个拆卸下来的尿兜算是什么。

走出特馆，走向常年的普通展览，里面的名画很多，很多人是冲着梵高的向日葵而去，贴心的地图还标明了热门名画的展出位置。可以见到许多学生或是热爱艺术的人面对大师们的作品临摹。用手机拨打标签上的号码便可以听到讲解，可以看到名贵的真迹在画布前都有一面玻璃，并且巡逻的管理人员常常会在附近走动。而我主要是看一些画中的美女，我不了解哪些是名画，哪些是名画家，单纯从养眼的角度，寻找画中的美女。这些图片都可以从博物馆的官方网站上找到，并且从官网上可以看到，费城最近在举办Inside Out 活动，把这些画的仿制品放在城市里合适的场景中，让你不期而遇。

{% imgcap http://thumbsnap.com/s/SzPvgOZ2.png Bend in the Epte River near Giverny -- Claude Monet %}

闭馆的时候天色已暗，可是还有半个多馆没有看完，在还没关门的礼品店挑好明信片之后，骑上回去的路，看来去宾大只能再挑一个下午了。自己不是一个有艺术造诣的人，只是有对艺术欣赏的好奇。

## 雨后天晴的上午

{% img right http://thumbsnap.com/s/6XGV2pG5.png 450 %}

{% img left http://thumbsnap.com/s/N3263YXF.png 450 %}
其实开会的时间选的相对比较运气，一连几天都没有感觉很冷，阳光都不错。然而今早下了点小雨，本来计划是去宾大的，骑车的话似乎要再三考虑了。最终还是决定骑车去，因为淋雨骑车，也是很棒的体验。

沿着Walnut St. 一路直行，很快就过桥，桥头写着欢迎来到宾大，就到了。另外说一句，这里东西向的街大多以吃的东西命名，什么Chestnut St.，Wine St.，而南北向的就直接是上顺序编号了。

骑着骑着，竟然不下雨了，算是老天给我了个面子。校园只有一个大致的范围，而没有明确的界限，里面的建筑和街上能见到的古迹没有太多的差异，唯一区别是市中心的建筑都偏高，加上狭窄的道路，让人感觉自己好渺小。而校园里基本都是教堂这么高的建筑，是一个很令人向往的地方，能在这里学习真是幸运。

停了车，跟着人流走进了一座教学楼。另说一句，骑车骑久了，也跟着这边的人一样，把车锁在管子上。选择了一个小教室坐了下来，期待蹭一节课听听。等待的时间里，渐渐犹豫，这么小的教室，上什么课都不清楚，万一听不懂又不好意思在大家面前走出去。思考再三，还是离开了教室，想要来这里听课，就靠实力考过来吧，就算在读一次本科也无妨。

离开教室，跨上战车，继续沿着陌生的路前行。想去图书馆看看，来到馆前，看到需要ID 卡，还是继续前行。同样的，以后要搞到张卡。

学校里面竟然还有卖纪念品的Bookshop，看来瞄准游客的商机已久。边上有一家二手书店，环境超棒，感觉和图书馆一样。店主在里面养了一只肥猫，很安静，很沉着，走路不同于路边的猫。

我想，有先见是一种非常难得的能力，这样就会在高考的时候想清楚以后要去怎样的学校，在高中的时候想清楚一些事然后做好，在初中的时候分析清楚兴趣和特长，等等等等，总之就是为了现在和以后的遗憾后悔。

{% imgcap http://thumbsnap.com/s/6EHU9T3J.png 莱斯丽，喜欢睡觉和思考 %}

说到底，远见和先知都是为了不让自己在漫漫人生路上逐渐消磨。

## 消化不良的晚上

初次来到异国，难免水土不服，不同的食物导致消化不良也是正常的。房东见我不适，便提议叫上他的小伙伴，一起去外面走走。

费城是个非常特别的城市，连住在这的美国本地人也这么认为。这座城市的街道比较窄，所以几乎都是单行线，街边随处可见都是古建筑，它们有些还在运作，有些正在出售，有些就这样荒废着，静静地等待着。美国人很想得开，基本所有的行当营业时间都在早上9点到下午5点，所以要代购什么还真要在白天买好。

我们走的是老城区的街道，James 说这里只有当地人晚上会游荡，是他们的乐园，一般观光团队都不会到这里，因为没有非常值得提及的景点，也没有什么购物中心，甚至连自然风光也只是错落的小公园以及绿化。但是这里的人喜欢这里的生活，这里有热闹的小酒吧，风味的批萨店，小书店，非常普通的生活区布局。最主要的是他们的朋友都住在附近，而且此次散步主要是来他们的好友家坐坐聊聊。

没有参与过多的谈话，毕竟只是一个陌生人。他们很诧异我还没有女朋友，我很惊讶他们可以很简单赚钱去世界各地旅行。Devin 说欢迎来美国，每个人都有枪。后来Devin 还教了我许多追女生的技巧。就这样，逐渐肚子舒服了些。

## 回家

费城是座很美的城市，很独特，就像在生活着的博物馆，希望有机会再来。乘坐着慢悠悠的火车去往机场，穿越大半座老城，暮色已起，玻璃上浮现了略带迷惘的脸。再见，费城。
